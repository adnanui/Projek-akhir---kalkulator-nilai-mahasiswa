# Projek-akhir---kalkulator-nilai-mahasiswa
program kalkulator nilai mahasiswa menggunakan bahasa C. Program menerima input data mahasiswa dan nilai akademik, melakukan validasi nilai 0–100, menghitung nilai akhir berdasarkan bobot yang ditentukan, serta menentukan status lulus/gagal dan grade A–E sesuai dengan flowchart yang telah dirancang.
