Alur program dimulai dari input data mahasiswa berupa nama dan NIM, kemudian dilanjutkan dengan input nilai untuk setiap komponen. Setiap nilai yang dimasukkan akan dicek terlebih dahulu agar berada dalam rentang 0 sampai 100 dan memastikan input berupa angka. Jika input tidak sesuai, program akan meminta pengguna untuk memasukkan ulang nilai tersebut.

Setelah semua nilai valid, program menghitung nilai akhir berdasarkan bobot yang sudah ditentukan. Dari nilai akhir tersebut, program menentukan status kelulusan (lulus atau gagal) serta grade mahasiswa dari A sampai E sesuai dengan flowchart yang telah dirancang. Terakhir, program menampilkan hasil perhitungan secara lengkap dan terstruktur agar mudah dibaca oleh pengguna.
